CHAT APPLICATION

Web Development - Web Desing

Interactive chat application that allows users to join a chat room where they can send and receive messages and notifications. This application was created using Node.js and Express on the back-end, in addition I used the webSocket socket.io which allows the app to have real-time, bidirectional and event-based communication. On the client side I used the front-end JavaScript framework Vue.js to render the messages and notifications as components.
