ROKU FLASHBACK MULTIMEDIA WEB APPLICATION

Web Development

The Roku flashback is a project where a partner and I were tasked to design and developed a multimedia web application capable of streaming audio and video. This web application also provides the user with a login system, in addition, users are able to filter and select a movie, tv show, or audio based on user profile (adult or kids). For this project I worked as the front-end and back-end developer.

The Development

For the development part of this project I used the LAMP stack (Linux, Apache, MySQL and PHP) for the server side and for the client side I used Vue.js as my front-end framework. On the server side I used a RESTful API that would run the SQL queries and fetch the data selected by the user from the database.

On the front-end side, I used Vue Router to render the different components based on the user profile and input. The application is populated with content by making AJAX calls and retrieving data using the Fetch API.
